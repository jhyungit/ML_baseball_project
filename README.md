# 📊 MLB 선수 연봉 예측 프로젝트

## 📌 프로젝트 소개
ML을 활용하여 MLB 선수의 다양한 성적 데이터를 기반으로  
연봉을 예측하는 **회귀 분석 프로젝트**입니다.  
(T-academy 개인 프로젝트)

---

## 🎯 개발 목적
- 데이터 기반 예측 모델 구축 경험
- 머신러닝 회귀 모델 이해 및 활용
- 데이터 전처리부터 모델 평가까지 전체 파이프라인 경험

---

## 🛠 기술 스택
- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## 🔍 주요 과정
- API 및 크롤링을 통한 데이터 수집
- 데이터 전처리 및 결측치 처리
- EDA(탐색적 데이터 분석)를 통한 변수 관계 분석
- 머신러닝 회귀 모델 학습 및 예측
- 모델 성능 평가 및 결과 분석

---

## ⚙️ 구현 방법

### 1. 데이터 수집
- API 및 웹 크롤링을 통해 MLB 선수 데이터 확보

### 2. 데이터 전처리
- 결측치 처리 및 불필요한 변수 제거
- 모델 학습을 위한 feature 구성

### 3. EDA (탐색적 데이터 분석)
- 주요 변수 간 상관관계 분석
- 연봉에 영향을 주는 주요 지표 확인

### 4. 모델 학습
- 회귀 모델을 활용한 연봉 예측
- 다양한 feature 조합을 통한 성능 개선 시도

### 5. 모델 평가
- 예측값과 실제값 비교를 통한 성능 평가

---

## 📊 결과 및 성과
- 선수 성적 데이터를 기반으로 연봉 예측 모델 구현
- 데이터 분석부터 모델링까지 전체 ML 프로세스 경험

- 결과 분석 및 시각화
